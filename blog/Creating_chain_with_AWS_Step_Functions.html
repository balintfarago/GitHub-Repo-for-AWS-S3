Creating a chain with AWS Step Functions

  • Photography
  • Balint Farago
  • 22nd March, 2023
Photography

In this example we’re going to do something very exciting. Step Functions can be perfectly used for this use case. Image a flow, where you add a product to the shopping cart, then you provide your address, then make the payment and finally you’re shown a ‘Thank you for your payment’ message. Let’s do this!

Let’s start with creating the first part of the flow. We’re going to create a Step Function that invokes a Lambda Function with 2 parameters, product name and product price and waits for the response. We’re also going to create a Lambda function that will receive the value of the product name and product price variables from the Step Function and will return these together with a product id, which is generated by the Lambda function.

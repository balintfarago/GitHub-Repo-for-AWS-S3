AWS services I used to create this site

On my journey to transition into cloud projects I have not only gained Azure and AWS certifications, but decided get hands-on experience by creating this website with AWS services.

Route 53 / S3

I purchased the this domain with Amazon Route 53 and set up Amazon S3 to host this website.

Amazon CloudFront

I am using CloudFront to distribute this site to edge locations to deliver content with low latency.

AWS Certificate Manager

I provisioned a public TLS certificate to protect this site.

AWS CodePipeline

I connected Visual Studio to Github and created a pipeline with AWS CodePipeline so all changes in the code are automatically pushed to this site.

AWS Lambda / SES

I set up a Lambda function to move uploaded .xml invoice files into separate folders based on the department being invoiced (which is an xml tag), after users upload them to a designated folder in an S3 bucket. Then I set up another function to send an email via AWS SES every time an invoice is uploaded to the bucket.

AWS Step Functions

I used Step Functions to invoke several Lambda functions in a purchase flow (add to cart, make payment, thank you) and pass variables among the different steps.
